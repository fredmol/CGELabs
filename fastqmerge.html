FastQ Merge

Combine multiple FastQ files into a single output

About This Tool

This tool allows you to merge multiple FastQ files:

  • Select a folder containing multiple FastQ files
  • Provide a name for the merged output
  • All compatible files in the folder will be merged automatically
  • The output will be saved as a single FastQ file
